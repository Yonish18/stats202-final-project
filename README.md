# Query-URL Relevance Prediction

This repository is being cleaned up into a recruiter-facing machine learning project structure.

The current codebase preserves the existing project substance without adding new experiments or unsupported claims. The intended project story, consistent with the final report, is:
- preprocessing and feature engineering
- Logistic Regression baseline
- HistGradientBoosting model
- final blended submission

Current layout:
- `src/` contains the active project scripts
- `results/` is for generated submissions and validation artifacts
- `report/` is reserved for the final PDF report and figures
- `archive/` retains earlier Random Forest and Gradient Boosting baseline work

Expected raw data files:
- `data/raw/training.csv`
- `data/raw/test.csv`

The README will be expanded later with setup details, report links, and a concise summary of the final results.
